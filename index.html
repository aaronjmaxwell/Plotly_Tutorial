An Introduction to Plotly

In order to follow this web booklet you need the local version of plot.ly (version 2.1.0). It can be installed through pip.

Scatter Plots

In Example 1: Boilerplate we create the boilerplate plot.ly JavaScript library by building two simple scatter plots in python.
In Example 2: Styling we add some styling to the scatter plots. We modify the marker annotation, sizes, opacity, and colors (and even add a color bar).

Line Plots

In Example 3: Lines we show a couple of neat options that plot.ly has implemented that requires significantly more work in matplotlib.
